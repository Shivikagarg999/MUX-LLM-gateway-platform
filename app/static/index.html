Mux

A multi-provider LLM gateway with automatic failover, semantic response caching, and per-key rate limiting — built on free-tier APIs only.

  • OpenAI-compatible /v1/chat/completions endpoint
  • Automatic failover: Groq → Gemini on any provider error
  • Semantic caching with local embeddings — similar prompts skip the LLM call entirely
  • Per-key auth, token-bucket rate limiting, and usage metering via /stats
